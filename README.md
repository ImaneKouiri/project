# Système de Gestion de Bibliothèque

## Architecture
- Java 17
- JavaFX
- Maven
- MySQL
- JDBC
- Modular project

## Infrastructure
- Singleton DatabaseConnection
- External configuration via config.properties
- Centralized MySQL access

## How to run
1. Start MySQL
2. Create database using create_database.sql
3. Configure config.properties
4. Run the application
